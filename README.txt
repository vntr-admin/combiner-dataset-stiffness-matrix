This trace set represents a stiffness matrix for an automobile chassis.  It has 35,588 vertices, 572914 edges, an assortivity of 0.4682, and partition removals at 984 and 9378.

It was provided by Davis and Hu via the Sparse Matrix Collection.

The University of Florida Sparse Matrix Collection, T. A. Davis and Y. Hu, ACM Transactions on Mathematical Software, Vol 38, Issue 1, 2011, pp 1:1 - 1:25. http://www.cise.ufl.edu/research/sparse/matrices.
